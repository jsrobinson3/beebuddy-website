## Features

- **Apiary & Hive Management** — Organize your bee yards and track individual colonies
- **Inspection Logging** — Record queen status, temperament, brood patterns, and more
- **AI Recommendations** — Provider-agnostic LLM integration (local Ollama or cloud APIs)
- **Photo Attachments** — Document hive conditions with images
- **Multi-User** — Secure authentication with social login support
- **Offline-Ready** — Designed for use in the field with sync when connected
- **Self-Hostable** — Docker Compose setup for easy local or server deployment
